Frequently Asked Questions relating to your Iriver

Sound quality is not satisfactory.

You may feel that the sound quality is not satisfactory under the following conditions.

  • 1. Objective reason:
  • If the MP3 file is created from low-level of audio features. Even the same MP3 file can have a different sound quality depending on the recording method.
  • MP3 file use the loss compression method. It is normal that MP3 has inferior quality than the general audio file. Higher bit rate of MP3 file or WMA file give better output sounds than the lower bit rate of them. But high bit rate files are using much more power than the low bit rate one.
  • 2. Subjective reason:
  • Each individual has a different criterion for the sound quality. Some people like strong bass while others don't. Measurement of the sound quality can be described by the S/N mechanically. The maximum S/N for CD recording is 90dB. iriver supports 90dB S/N, which ensures the sound output that is almost corresponds to the recording quality.

Song title is not displayed but the audio track number is displayed.

iriver can display the song title information on the LCD only when the file contains it. The MP3 file can contain the wrong tag contents, or the normal audio CD can be displayed in such a way in iMP model since no song information is available. To correct this problem, edit the tag information in the MP3 file, or burn the audio CD after setting the CD-text function in the iMP model.

What is a tag? How does it to be displayed?

A tag is the format that saves the title, artist and album information in the MP3 file and displays it on the MP3 player. iriver supports up to 60 characters (including space character) regardless two bytes or one byte languages. If you set Tag display off, the file name will be displayed.

If I insert an adaptor while running the player with a battery, the player is turned off.

iriver is designed to shut the power off if the adaptor is inserted for the purpose of protecting the battery lifespan. On the contrary, if you disconnect the adaptor while running the player with an adaptor, the player will also be turned off. Sometimes, the player may not be turned off due to the circuit charging status. We recommend you to turn the power off using the Stop key when connecting or disconnecting the adaptor, or press the Play key after connecting or disconnecting the adaptor. If you set the Resume On, playback back will start from the stopped position.

Radio reception is too poor.

Radio reception sensitivity varies according to the area, and certain level of interference is expected on the move. Especially, radio reception can fail in the weak frequency sensitivity area. The car stereo or the general component system uses the fixed antenna. Therefore, it can catch the radio frequency more sensitively than the portable device like iriver that uses the earphone (remote controller) as an antenna. If you think the player provides inferior radio reception sensitivity than other portable radio products, please contact the shop where you get the player.
